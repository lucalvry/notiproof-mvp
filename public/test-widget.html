Widget Installation

Add this script tag to your website to enable NotiProof widgets:

<script src="https://your-domain.com/widget.js" data-widget-id="YOUR_WIDGET_ID"></script>

Note: Replace "YOUR_WIDGET_ID" with your actual widget ID from the dashboard.
